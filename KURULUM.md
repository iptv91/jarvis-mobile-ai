# 🤖 JARVIS - Kurulum Rehberi

## 📱 Telefonda Kurulum

### Adım 1: APK'yı İndir
```bash
jarvis-release.apk
```

### Adım 2: Bilinmeyen Kaynaklar'ı Etkinleştir
1. **Ayarlar** → **Güvenlik**
2. **Bilinmeyen Kaynaklar** → **Aç** ✅

### Adım 3: APK'yı Kur
1. Dosya Yöneticisi'nde `jarvis-release.apk`'yi bulun
2. Üzerine tıklayın
3. **Kur** butonuna tıklayın

### Adım 4: İzinler
Uygulamayı açın ve gerekli izinleri verin:
- ✅ Telefon
- ✅ SMS
- ✅ Kamera
- ✅ Mikrofon
- ✅ Dosyalara Erişim

---

## 💻 Bilgisayarda Build

### Gereksinimler
- Node.js 16+
- npm veya yarn
- 16GB RAM
- 30GB Boş Disk

### Kurulum

```bash
# 1. Repo'yu klonla
git clone https://github.com/iptv91/jarvis-mobile-ai.git
cd jarvis-mobile-ai

# 2. Setup script'i çalıştır
bash build-setup.sh

# 3. APK dosyası Al
ls jarvis-release.apk
```
